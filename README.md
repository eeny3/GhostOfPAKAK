# GhostOfPAKAK
arena based survival game

Play as a powerful ghost and survive as many waves as possible.

Controls:
- WASD for movement
- Q + <moue_position> for electric sphere
- W + <mouse_pos> for AOE frostbolt
- E + <mouse_pos> for AOE hellfire
